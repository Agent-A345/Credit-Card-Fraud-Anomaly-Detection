## 🚨 Credit Card Fraud Anomaly Detection

A machine learning project built using Python and Jupyter Notebook that detects anomalous and potentially fraudulent credit card transactions using unsupervised anomaly detection techniques. The project demonstrates data preprocessing, anomaly detection, transaction analysis, and visualization using real-world financial transaction data.

## 🚀 Features
* **Anomaly Detection System** – Identify unusual credit card transactions automatically.
* **Fraud Detection Analytics** – Detect suspicious financial activities without labeled data.
* **Unsupervised Machine Learning** – Apply anomaly detection algorithms for fraud analysis.
* **Financial Transaction Analysis** – Analyze customer transaction behavior and anomalies.
* **Data Cleaning & Preprocessing** – Prepare transaction data for anomaly detection.
* **Exploratory Data Analysis (EDA)** – Visualize transaction distributions and fraud patterns.
* **Anomaly Visualization** – Generate plots highlighting suspicious transactions.
* **Feature Analysis** – Explore transaction characteristics influencing anomalies.
* **Performance Evaluation** – Assess anomaly detection effectiveness using evaluation metrics.
* **Real-World Financial Dataset** – Practice anomaly detection using authentic banking data.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Credit-Card-Fraud-Anomaly-Detection.git
```
2. **Navigate to the project folder**:
```
cd Credit-Card-Fraud-Anomaly-Detection
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter scikit-learn 
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
anomaly_detection.ipynb
```
